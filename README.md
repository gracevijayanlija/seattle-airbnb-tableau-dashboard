# Seattle Airbnb Market Analysis — Tableau Dashboard

Analyzed 3,818 Seattle Airbnb listings to uncover pricing trends by location and property size, and visualize booking activity across 2016.

![Dashboard Overview]

**Tools:** Tableau Desktop, Excel

**What's in the dashboard (5 views):**
- Avg price by zipcode (bar + filled map)
- Avg price by bedroom count
- Weekly price trend across 2016
- Listing count by bedroom count

**Key insights:**
- Avg price: **$127.98** (range $20–$1,000) across 28 zipcodes
- Price scales with bedroom count: $95.71 (1BR) → $441.00 (5BR)
- 1BR listings are the most common (63% of all listings) but priced below studios

**Data note:** Calendar sheet hit Excel's row limit during export, so it covers ~75% of listings (2,873 of 3,818). Flagging this for transparency.

**Files:** `tableau_full_project_airbnb.twb` (open in Tableau Desktop/Reader)
**Dataset:**The dataset is publicly available from [https://www.kaggle.com/alexanderfreberg/airbnb-listings-2016-dataset]. Due to GitHub file size limits, it is not included in this repository.
